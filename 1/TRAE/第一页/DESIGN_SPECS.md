# 设计参数对照文档

## 颜色
- 主色: `--primary-color: #007bff`
- 文本: `--text-color: #333333`
- 背景: `--bg-color: #ffffff`

## 字体
- Base family: `-apple-system, BlinkMacSystemFont, Segoe UI, Roboto, Helvetica Neue, Arial, sans-serif`
- 行高: `1.6`
- 标题粗细: `600`

## 间距
- 基准单位: `--spacing-unit: 8px`
- 常用间距: `mt-1/2, mb-1/2, py-1/2`

## 圆角
- 小: `--radius-sm: 6px`
- 中: `--radius-md: 10px`
- 大: `--radius-lg: 16px`

## 阴影
- 轻: `--shadow-soft: 0 4px 12px rgba(0,0,0,0.08)`
- 重: `--shadow-hard: 0 12px 28px rgba(0,0,0,0.16)`

## 栅格系统
- 容器宽度: `max-width: 1200px`
- 行: `.row`，列: `.col`
- 列宽: `.col-12/.col-6/.col-4`
- 响应: `.col-md-6/.col-md-4/.col-md-3` (≥768px)
